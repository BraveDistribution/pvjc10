Lab 10: Basics of Electronics

Programming in C — PVJC — FEI TUKE

The goal of this lab is to get familiar with basic electronic components and understand their behavior in simple circuits. You will use Arduino Uno connected via USB as a power source (pins +5V and GND). No programming today, we work purely with hardware.

Available components

  • Arduino Uno + USB cable
  • Breadboard
  • LED diodes (various colors)
  • Resistors (various values)
  • Potentiometer
  • Push buttons (4-pin)
  • Jumper wires

Setup

  1. Connect Arduino Uno to the computer via USB cable.
  2. Wire +5V pin from Arduino to the red rail (+) on the breadboard.
  3. Wire GND pin from Arduino to the blue rail (-) on the breadboard.

Rails (+/-) are connected longitudinally, rows are connected horizontally in groups of 5.

Warning: Never connect an LED without a series resistor. Without one, the LED can be destroyed instantly!

1 Logic Gates from Buttons

Background

In digital logic, there are basic operations AND and OR:

  • AND: output is active only when both inputs are active simultaneously.
  • OR: output is active when at least one input is active.

Tasks

  1. Calculate the appropriate series resistor value for an LED:
    • Source voltage: 5V
    • LED voltage drop: ~2.0V
    • Required current: 20mA
    R = (Vsource − Vled) / I
  2. Build an AND circuit: Wire two buttons in series with an LED and resistor. LED lights only when both buttons are pressed.
  3. Build an OR circuit: Wire two buttons in parallel with a different LED and resistor. LED lights when either button is pressed.
  4. For each circuit, fill in the truth table:

11??

Questions

  1. Why does this wiring behave as a logic gate?
  2. Can you build a NOT gate using only buttons and LEDs? Why or why not?

2 Series vs. Parallel LED Circuits

Background

LEDs can be wired in series (one after another) or in parallel (side by side). Each configuration has a different effect on voltage distribution and current flow.

Tasks

  1. Wire 2 LEDs in series with a single resistor. Calculate the required resistor value, considering voltage drop is now 2× ~2.0V.
  2. Wire 2 LEDs in parallel with a single resistor. Consider what current flows through the resistor if each LED should carry 20mA.
  3. Compare brightness of LEDs in both configurations. Record your observations.
  4. Try wiring 3 LEDs in series. What happens and why?

Questions

  1. Why does LED brightness differ between series and parallel configurations?
  2. What is the maximum number of LEDs you can wire in series with 5V supply? Justify with a calculation.
  3. What are the advantages and disadvantages of each configuration in practice (e.g., Christmas lights)?

3 Adjustable Circuit with Potentiometer

Background

A potentiometer is a variable resistor with three pins. Rotating the shaft changes the resistance ratio between the outer and middle pin. This allows smooth regulation of voltage and current.

Tasks

  1. Wire the potentiometer as a variable resistor in series with an LED and a fixed resistor (220Ω):
    • One outer pin to +5V
    • Middle pin (wiper) to LED anode
    • Other outer pin disconnected
  2. Rotate the potentiometer and observe LED brightness change. Note at which position the LED turns off and at which it is brightest.
  3. Add a button so the LED only lights when pressed, while brightness is still controlled by the potentiometer.
  4. Bonus: Wire the potentiometer as a voltage divider: connect both outer pins to +5V and GND, middle pin through a resistor to LED.

Questions

  1. Why is it important to have a fixed resistor alongside the potentiometer?
  2. What would happen to the LED if the potentiometer was at zero resistance with no fixed resistor?

Additional Tasks

For students who finish early:

  1. Build a circuit where one button switches between two LEDs (red/green). Hint: the button has 4 pins.
  2. Combine all three experiments: two buttons (AND) control a circuit with two parallel LEDs whose brightness is regulated by a potentiometer.
